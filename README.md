# Terminal-based Tetris Engine
A robust implementation of Tetris built in C++ using the ncurses library. This project serves as a demonstration of state-machine architecture and real-time event handling.

Key Technical Features:

State-Machine Architecture: Implements a robust loop for managing game states (Pause, Hold, Drop, Game Over).

Data Structure Proficiency: Utilizes std::vector for dynamic grid and color management.

Real-time Rendering: Employs ncurses for non-blocking terminal UI updates and high-performance frame rendering.

Modularity: Features a decoupled system for piece rotation, line clearing, and hard-dropping mechanics.
